# NeuroWeave 🧠

NeuroWeave is an AI-powered platform that connects scattered information and transforms it into a visual knowledge network. It helps users discover relationships between ideas and understand complex topics faster.

## Features
- AI-powered concept relationship detection
- Knowledge graph visualization
- Smart information organization
- Interactive exploration of ideas

## Tech Stack
- React
- Node.js
- Python
- OpenAI API
- Network Graph Visualization

## How it works
1. User enters a topic or information
2. AI analyzes relationships between concepts
3. The system builds a knowledge graph
4. Users explore connected ideas visually
